## 7장, 카프카를 활용한 데이터 파이프라인 구축
---
- 7.1 카프카를 활용한 데이터 흐름도
    - 프로듀서
        - 메세지 생산 후 카프카로 전송
    - 카프카
        - 생산한 메세지를 저장하는 중간 큐 역할
    - 컨슈머
        - 카프카에 저장된 메세지를 가져옴
        - 데이터 처리, 로그 분석을 위해
            - 다른 어플리케이션으로 메세지를 전송하는 역할
    - 흐름도 및 기술 스택
        - 프로듀서 : Filebeat
        - 컨슈머 : Nifi
        - 데이터 처리 : Nifi
    - Nifi
        - Apache Nifi
        - 데이터 흐름을 정의하고, 정의된 흐름대로 자동으로 실행해주는 어플리케이션
        - 데이터 흐름 기반으로 사용하기 유용
        - 웹 기반 인터페이스 제공
